Type WHO USES IT? WHAT MAKES LEARNING IT IMPORTANT?
HTML Web developers, technical editors, email designers, software engineers. HTML is used to create electronic documents (pages) displayed online. Visit any page and you will see an example of HTML in action.
JavaScript JavaScript developers, Web developers, software engineers. JavaScript is used primarily in Web development to manipulate various page elements and make them more dynamic, including scrolling abilities, printing the time and date, creating a calendar and other tasks not possible through plain HTML. It can also be used to create games and APIs.
